# PyCMS

## Software Information

1. Version: 0.1
2. Description: A headless CMS built by Django and VueJS
3. Main features: manage pages, manage posts and categories

## Development

```sh
source .venv/bin/activate
uv .venv
uv add Django

django-admin help
django-admin startproject cms .
```
